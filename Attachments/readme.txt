1. EncryptSecret 
	- Use this program for encrypting the Shared Secret for SCP
	- Program uses a file named "CardKey.txt" from your "Documents" folder
	- Encrypted Shared Secret is also stored in your "Documents" folder
2. Authentication Applet
	- To install this applet to your Java Card, use the GPShell tool
	- The CAP file is in the "bin" folder
3. JavaCardPlugin
	- Move this entire folder to "..\KeePass Password Safe 2\Plugins"
	- no additional action is required (in case you have encrypted your Shared Secret)
